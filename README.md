# 🤖 Quantum Bot (양자컴퓨터 AI 퀀트 자동매매 봇)

[English](README_EN.md) | [한국어](README.md)

![License](https://img.shields.io/badge/license-MIT-green.svg)
![Toss API](https://img.shields.io/badge/Toss_Invest-Open_API-0050FF.svg)

Quantum Bot은 토스증권 API와 텔레그램을 연동하여, **양자컴퓨터 관련 미국 주식 5종목**의 비중을 자동으로 관리하고 월스트리트 수준의 퀀트 알고리즘으로 매수/매도 타이밍을 분석해주는 개인용 인공지능 비서입니다.

---

## 🎯 타겟 포트폴리오 (Target Portfolio)

로봇은 아래 5개의 핵심 양자컴퓨터 기업을 타겟으로 하여, 사용자가 설정한 황금 비율(Target Weight)을 무한히 유지합니다.

*   🔵 **IONQ (아이온큐)** - 35%
*   🔵 **INFQ (인플렉션)** - 25%
*   🔵 **QBTS (디 웨이브 퀀텀)** - 15%
*   🔵 **RGTI (리게티 컴퓨팅)** - 15%
*   🔵 **QUBT (퀀텀 컴퓨팅)** - 10%

*(설정된 비중은 `config.py`에서 언제든지 자유롭게 변경할 수 있습니다.)*

## 🌟 주요 기능 (Core Features)

1. **가상 지갑(Virtual Wallet) 기반 자정 리밸런싱**: 매일 자정(KST), 설정해 둔 목표 비중에 맞게 자동으로 주식을 사고팔아 밸런스를 맞춥니다. **가상 지갑 시스템**을 도입하여 계좌의 예수금을 0원으로 유지하면서도, 소수점 매매 한계로 인해 발생하는 짜투리 달러를 스스로 기록 및 보관하여 포트폴리오 자산 축소(디플레이션)를 완벽하게 방지합니다.
2. **이중 API 안전장치 & 자가 치유(Self-Healing)**: 토스증권 서버에서 계좌의 실제 예수금을 사전 조회하여 잔고가 부족하면 스스로 주문을 차단합니다. 또한 사용자가 직접 출금을 하여 장부상 현금과 실제 현금에 오차가 생기면 봇이 이를 감지하고 스스로 가상 지갑 잔고를 깎아 동기화(Self-Healing)합니다.
3. **토큰 자동 갱신 및 에러 복구**: 토스증권 API 토큰 만료(401)나 잦은 요청으로 인한 과부하(429)가 발생할 경우 스스로 새 토큰을 발급받고 재시도하는 강력한 생존 로직이 탑재되어 있습니다.
4. **다중 통화(Multi-Currency) 시황 브리핑**: 환율, 포트폴리오 상태, 종목별 퀀트 점수, 최신 구글 뉴스를 하루 4번 요약 보고합니다. 내 계좌 현황에서 **한국 주식(₩)과 미국 주식($)**의 자산 및 당일 수익을 명확하게 분리해서 보여줍니다.
5. **장중 실시간 감시**: 미국장이 열려있는 동안 5분 간격으로 차트를 감시하여, 강력 매수/매도 타점 발생 시 텔레그램으로 긴급 시그널 알림을 쏩니다. (스팸 방지를 위해 자동 매매는 하지 않으며 알림만 발송)
6. **미국장 완벽 달력 연동(휴장일 회피)**: `holidays` 라이브러리를 통해 시차를 반영, **미국장 및 한국장 공식 휴장일(독립기념일, 추수감사절 대체휴일 등)과 주말**을 스스로 파악하여 모든 알림과 리밸런싱을 똑똑하게 건너뛰고 휴식합니다. 반쪽짜리 조기 종료(Half-day) 장에서도 완벽하게 동작합니다.
7. **글로벌 타임존 고정 (KST)**: 노트북을 들고 해외로 출국하여 시스템 시간이 바뀌더라도, 봇은 내부적으로 **한국 표준시(KST, Asia/Seoul)**를 굳건히 유지하여 스케줄 오차를 방지합니다.

## ⚙️ 설치 및 세팅 방법 (Getting Started)

### 1. 사전 준비 (Prerequisites)
* Python 3.9 이상
* 토스증권 실전/모의투자 Open API 키 및 시크릿 키
* 텔레그램 봇 토큰 및 Chat ID

### 2. 패키지 설치
```bash
git clone https://github.com/USERNAME/quantum_bot.git
cd quantum_bot
pip install -r requirements.txt
```
*(참고: `requests`, `schedule`, `holidays`, `beautifulsoup4` 등의 패키지가 필요합니다.)*

### 3. 환경 변수 설정 (`.env`)
프로젝트 루트 폴더에 `.env` 파일을 생성하고 아래 내용을 본인의 정보로 채워주세요.

```ini
# 토스증권 API 접근 정보
TOSS_API_KEY=your_api_key_here
TOSS_SECRET_KEY=your_secret_key_here

# 텔레그램 봇 알림 설정
TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here
TELEGRAM_CHAT_ID=your_chat_id_here
```

### 4. 설정 파일 확인 (`config.py`)
`config.py` 파일에서 포트폴리오 타겟 비중 및 실전 투자 모드 여부를 설정할 수 있습니다.
* `PAPER_TRADE = True`: 모의투자 모드 (실제 매매 차단, 텔레그램 알림만 발송)
* `PAPER_TRADE = False`: 실전 투자 모드 (실제 계좌에서 매수/매도 발생)

## ⏰ 24시간 스케줄 흐름 (Total Workflow)

터미널에 봇을 띄워두면 아래의 시간표(**무조건 한국 시간 기준**)에 맞춰 전자동으로 작동합니다. *(주말 및 공휴일에는 봇이 자동으로 휴식합니다)*

*   `08:30` - 🇰🇷 한국장 개장 전 브리핑 (평일 한정)
*   `15:40` - 🇰🇷 한국장 마감 브리핑 (평일 한정)
*   `22:00` - 🇺🇸 미국장 개장 전 브리핑 (월~금, 미국 휴장일 제외)
*   **`22:30 ~ 05:00` - 🇺🇸 장중 실시간 시그널 감시 (5분 간격)**
*   **`00:00` - 🔄 자정 자동 리밸런싱 (화~토, 미국 휴장일 제외)**
*   `05:10` - 🇺🇸 미국장 마감 브리핑 (화~토, 미국 휴장일 제외)

## 🧠 퀀트 분석 알고리즘 (Scoring Engine)
봇은 과거 100일 치의 캔들 데이터를 수학적으로 분석하여 100점 만점의 매매 스코어를 채점합니다.
1.  **다중 이동평균선 (10점)**: 5일, 20일, 50일선의 정배열/역배열 판별
2.  **볼린저 밴드 (20점)**: 표준편차 계산을 통한 과열 및 과낙폭 판별
3.  **MACD & 시그널선 (20점)**: 지수이동평균(EMA)을 활용한 추세 모멘텀 판별
4.  **RSI (20점)**: 14일 상대강도지수를 통한 과매수/과매도 판별
5.  **내 평단가 대비 수익률 (30점)**: 물타기(저가매수) 및 수익 실현 타이밍 점수 부여

## 💻 스마트 매매 명령어 사용법 (Manual Trade)
추가로 현금을 투입하거나 일부 출금이 필요할 때 터미널에 아래 명령어를 치면, 봇이 부족/초과분을 똑똑하게 계산해서 매수/매도해 줍니다.

*   **2천 달러를 추가로 투자(매수)할 때:**
    ```bash
    python3 trade_cash.py buy 2000
    ```
*   **5백 달러어치 현금이 필요해서 일부 매도할 때:**
    ```bash
    python3 trade_cash.py sell 500
    ```

## 🚀 봇 실행 방법 (How to Run)
봇이 24시간 동안 작동하게 하려면 백그라운드 구동 명령어(`nohup`)를 사용합니다. **(맥북의 경우 클램쉘 모드와 Amphetamine 앱 사용을 권장합니다)**

```bash
# 백그라운드 모드로 봇 실행
nohup python3 main.py > nohup.out 2>&1 &

# 로그 실시간 확인
tail -f nohup.out

# 봇 강제 종료
kill $(pgrep -f "main.py")
```
